## Contributing to Documentation

### Documentation Standards

All documentation should follow:
- **[Style Guide](meta/STYLE_GUIDE.md)** - Formatting and style standards
- **[Templates](templates/)** - Use appropriate templates for new docs
- **[Maintenance Guidelines](meta/MAINTENANCE.md)** - Keep docs current

### Creating New Documentation

1. Choose appropriate template from `templates/`
2. Follow style guide standards
3. Add to this index
4. Update related documentation with cross-references
5. Submit PR for review

### Updating Existing Documentation

1. Follow style guide
2. Update "Last Updated" date
3. Verify all links work
4. Update related docs if needed
5. Submit PR for review
